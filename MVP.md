# 退休金测算小程序 — MVP 需求文档

## 1. 产品定位

**一句话**：输入几个参数，30秒算出你退休后每月能拿多少钱。

**目标用户**：
- 25-50岁在职人员，关心退休规划
- 即将退休人员，想提前了解收入预期
- HR/财务人员，帮员工做规划

**核心差异化**：
- 支持全国主要城市
- 可视化展示，不只是一个数字
- "多缴一年多多少"的直观对比

---

## 2. 养老金计算公式

### 基础养老金
```
基础养老金 = 退休时当地上年度在岗职工月均工资
           × (1 + 本人平均缴费指数) ÷ 2
           × 缴费年限 × 1%
```

### 个人账户养老金
```
个人账户养老金 = 个人账户累计储存额 ÷ 计发月数
```

计发月数（按退休年龄）：
- 50岁退休：195个月
- 55岁退休：170个月
- 60岁退休：139个月
- 65岁退休：101个月

### 个人账户累计储存额
```
每月个人缴费 = 缴费基数 × 8%
年利率：按国家公布记账利率（约6%，简化处理）
```

### 总月养老金
```
月养老金 = 基础养老金 + 个人账户养老金
```

## 3. 用户输入参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 出生年份 | 用于计算距退休年限 | 1985 |
| 性别 | 影响法定退休年龄 | 男 |
| 参保城市 | 影响社平工资基数 | 北京 |
| 开始缴费年份 | 计算缴费年限 | 2010 |
| 月均缴费基数 | 实际工资或选择倍数 | 社平工资×1 |
| 预计退休年龄 | 60/55/50（按政策） | 60 |

---

## 4. 功能范围（MVP）

### 必须有
- 参数输入表单（滑块+输入框）
- 计算结果展示：月养老金、年养老金
- 基础养老金 vs 个人账户养老金拆分展示
- 柱状图：不同退休年龄对比（55/60/65岁）
- 主要城市社平工资数据（20个城市）

### 付费解锁（¥9.9）
- 多缴费年限对比（再缴5年/10年多多少）
- 详细计算过程报告（可截图分享）
- 断缴影响测算

### 不做（MVP 范围外）
- 企业年金计算
- 职业年金
- 商业保险对比

---

## 5. 页面结构

```
首页（输入页）
  ↓
结果页（展示月养老金 + 图表）
  ↓
详情页（付费解锁，详细报告）
```

## 7. 技术方案

```
框架：    微信小程序原生（WXML + WXSS + JS）
图表：    wx-charts 或 echarts-for-weixin
计算：    纯前端 JS，无需后端
数据：    城市社平工资存本地 JSON
支付：    微信支付（付费解锁功能）
```

---

## 8. 变现设计

- 免费：基础测算，结果页展示月养老金
- 付费 ¥9.9：解锁详细报告 + 多方案对比
- 广告：结果页底部展示微信广告（流量主）
